Local 2-node practice cluster
The practice tasks assume a real cluster (one control-plane, one worker). This one-command toolkit spins up a kubeadm-based kind cluster on your machine — with Calico (NetworkPolicy), metrics-server, and ingress preinstalled — and resets to a clean state in seconds. Works on Linux, macOS, and Windows. No repo clone needed.
⬇️ Download for Linux / macOS ⬇️ Download for Windows
…or grab it straight from your terminal:
Linux / macOS curl -fsSL https://bartekmp.github.io/cka-learning-tracker/cluster/get.sh | bash Copy
Windows irm https://bartekmp.github.io/cka-learning-tracker/cluster/get.ps1 | iex Copy

Requires Docker running. kind and kubectl are installed automatically if missing. Then run ./setup.sh (or ./setup.ps1) and reset any time with ./reset.sh. Full details in the cluster README ↗.

This is optional. If you can't run a cluster locally (no Docker, low-spec machine, locked-down work laptop), you don't need it — do the tasks in the free, browser-based Killer Coda CKA labs ↗ instead. The two kubeadm tasks (cluster upgrade, bootstrap-from-scratch) are better done there anyway.
